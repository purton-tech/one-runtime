What your agents can ask One Runtime

list_integrations()

Return the integrations and tool families currently exposed through One Runtime.

list_functions("gmail")

Return callable Python functions for a specific integration.

get_connection_status("gmail", user_id)

Tell the model whether the function is ready, needs a user connection, or is blocked on an upstream key.

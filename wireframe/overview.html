How agents use One Runtime

View explainer

1. Discover tools

The model can search One Runtime for integrations, inspect exposed Python functions, and see whether the capability is ready.

2. Check connection state

Before calling Gmail or another provider, the model can see whether a user connection exists or whether an upstream API key is still missing.

3. Call or remediate

If connected, the model calls the function. If not, it can return a hosted user link or ask an operator to add the missing provider credential.
